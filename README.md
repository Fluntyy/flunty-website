Once you've ~~created a~~ stole **MY** project and installed dependencies with `npm install` (or `pnpm install` or `yarn`), start a development server:

```bash

npm run dev -- --open
```

## Build it

To create a production version of **MY WEBSITE**:

```bash
npm run build
```
